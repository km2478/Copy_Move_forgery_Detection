# Copy_Move_forgery_Detection

Thank you for visiting in my project. The project name it self says the project is about to find the fake image and real image. 
Forgery says that any changes done in image or temped in image which are not there earlier. 
In our project we detect the forge and real image and we use ela to find out the fake and original details in the image.

Used 128 * 128 image size.
Image format is used: ".jpg", ".png", ".jpeg"
Dataset used from the kaggel. 
Total number of images are something about 10 to 15 thousand. Using balanced process the duplication of the image is removed, so the model can run with good accuracy.
